class为构造函数的另一种写法，其主要操作在constructor()函数内

四件事：new Object创建对象，自动设置继承关系，this指向对象并赋值，返回新的对象

class必须配合new使用，否则报错。而构造函数不使用new也不会报错，返回undefined

class内this指向new后的实例，而使用static后，使用this则是指向class类本身而非实例

constructor函数内的this.name = name可以写在类的最上层去掉this,等同于写在constructor

静态属性与静态方法一样只能在class使用，不能供实例使用

class fn{ constructor(a='于立伟',b=12) { this.name = a; this.age = b } } function fn1(a='于立伟',b=12) { this.name = a; this.age = b; } var s = new fn1() class fn2{ name='秦媛丽' age=80 say() { console.log(`我叫${this.name},我${this.age}岁了`) } } let ss = new fn2()
